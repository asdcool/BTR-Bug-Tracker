# 🐞 BTR – Bug Tracking System

## 📌 Overview

BTR is a lightweight bug tracking application developed for a Software Engineering course project
It allows users to record, view, and manage software bugs through a simple and intuitive interface.

___

## 🚀 Features

### Core Functionality (Sprint 1)
- Record new bugs with detailed attributes:
  - Title, Description
  - Bug Type and Artifact Type
  - Reporter and Date Found
- View a list of all reported bugs
- Update bug lifecycle status:
  - Open → In Progress → Fixed → Closed

---

### 🆕 Enhancements (Sprint 2)
- Improved graphical user interface using Tkinter for better usability
- Editable bug records:
  - Modify existing bug details through a popup window
- Enhanced layout and window management:
  - Responsive resizing and improved form alignment
- Persistent data storage using SQLite with automatic updates
- Input validation to prevent incomplete or incorrect entries
- Improved user interaction with clearer buttons and structured forms

---

### 🧪 Testing & Reliability
- Automated unit tests for key functionalities:
  - Bug creation
  - Status updates
  - Data retrieval
- Verified test results with all core features passing
