# 🐞 BTR – Minimal Bug Tracking System

📌 Overview

BTR (Bug Tracking Report) is a lightweight bug tracking application developed for a Software Engineering course project (Sprint 2).
It allows users to record, view, and manage software bugs through a simple and intuitive interface.

⸻

🚀 Features
	•	Record a new bug with detailed information
	•	View all reported bugs
	•	Update bug lifecycle status (Open → In Progress → Fixed → Closed)
	•	Persistent storage using SQLite database
	•	Simple and clean graphical user interface (Tkinter)
	•	Automated unit testing for core functionalities
